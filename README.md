## 🧠 Sobre o Projeto

- Projeto de uma ferramenta web para análise automática de textos.
- Permite inserir qualquer texto e gerar um relatório com estatísticas das palavras.
- Analisa frequência de palavras utilizadas.
- Exibe métricas básicas do texto.
- Interface simples e rápida para facilitar o uso.

O objetivo do projeto foi praticar conceitos importantes de desenvolvimento web, como:

- manipulação de strings
- lógica de contagem e análise de dados
- interação com o DOM
- construção de interfaces web simples

Também é um bom exercício para quem deseja aprender:

- análise básica de dados em JavaScript  
- criação de ferramentas web úteis  
- desenvolvimento de pequenos projetos para portfólio
